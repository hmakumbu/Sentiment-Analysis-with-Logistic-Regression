# Sentiment-Analysis-with-Logistic-Regression

```markdown

This project implements logistic regression for sentiment analysis on tweets. The objective is to determine whether a tweet has a positive or negative sentiment. 

## Key Features

- Extract features for logistic regression from text
- Implement logistic regression from scratch
- Apply logistic regression to a natural language processing task
- Test and validate your logistic regression model
- Perform error analysis
```
